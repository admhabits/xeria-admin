Laravel Version

Installation

The zip file contains all js files integrated with Xeria, however you need to perform following steps to get vendor etc.

Minimum Requirements
 
    For laravel 5.8 , the minimum version of php required is 7.1.3 and the following php extensions are required

    PHP >= 7.1.3
    OpenSSL PHP Extension
    PDO PHP Extension
    Mbstring PHP Extension
    Tokenizer PHP Extension
    XML PHP Extension
    Ctype PHP Extension
    JSON PHP Extension
Get Composer packages
 composer install
Permissions
 chmod -R 775 storage

 chmod 775 bootstrap/cache

If you are on linux/mac you can run below command to chown it.

 chown -R www-data /var/www

After composer installation create a new directory anywhere in your system for your new Laravel project. After that, move all files contains in the downloaded zip file.

You have to install node on your local machine or server to compile assets.

Note : Make sure this all installation process is done in command prompt

Run the following comands in command promt to start laravel development server.


    composer install
    npm install gulp-cli -g
    npm install gulp -D
    npm install
    npm install -g cross-env
    npm run dev
    php artisan serve
